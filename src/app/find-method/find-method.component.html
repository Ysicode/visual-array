What does the JavaScript find function do?

If the callback returns a truthy value, the find function immediately returns the element and stop searching. Otherwise, it returns undefined.

"This method returns the value of the first element in an array that pass the test in a testing function."

Code Example

let numbers = [1, 2, 3, 4, 5, 6, 7];

let found = numbers.find(number => number > 4);

console.log(found);

//Output = 5
