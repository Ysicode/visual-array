What does the JavaScript findIndex function do?

The findIndex method returns the index of the element that satisfies a testing function or -1 if no element passed the test.

"This method returns the index of the first element in an array that pass the test in a testing function."

Code Example

let numbers = [1, 2, 3, 4, 5, 6, 7];

let indexFound = numbers.findIndex(number => number == 4);

console.log(indexFound);

//Output = 3
