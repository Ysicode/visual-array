What does the JavaScript reverse function do?

"This method reverses an array in place. Element at last index will be first and element at 0 index will be last."

Code Example

let numbers = [5, 8, 3, 7, 1, 4, 2];

numbers.reverse();

console.log(numbers);

//Output = [2, 4, 1, 7, 3, 8, 5]
