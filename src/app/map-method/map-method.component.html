What does the JavaScript map function do?

The map method in JavaScript creates an array by calling a specific function on each element of a provided array. Generally the map method is used to iterate over an array and calling a function on every element of an array.

"This method creates a new array with the results of calling a provided function on every element in this array. "

Code Example

let numbers = [1, 2, 3, 4, 5, 6, 7];

let mapped = numbers.map(number => number + 20);

console.log(mapped);

//Output = [21, 22, 23, 24, 25, 26, 27]
