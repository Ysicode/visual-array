What does the JavaScript every function do?

The Javascript every method executes a callback function once for each element in the array. If you call the every method on an empty array, the method will always return true for any condition.

"This method checks every element in the array that passes the condition, returning true or false as appropriate."

Code Example

let numbers = [1, 2, 3, 4, 5, 6, 7];

let greaterFive = numbers.every(number => number > 5);

console.log(greaterFive);

//Output = false

let smallerOne = numbers.every(number => number > 0);

console.log(smallerOne);

//Output = true
