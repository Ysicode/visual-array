What does the JavaScript filter function do?

In JavaScript, the filter method allows us to filter through an array - iterating over the existing values, and returning only the ones that fit certain criteria, into a new array. The filter function runs a conditional expression against each entry in an array. If this conditional evaluates to true, the element is added to the output array. If the condition fails, the element is not added to the output array...
